# Configurations
## Settings
Settings are all json-files, put an a specific place with a specific filename:
- workspace
- user

Examples:
- Zoom
- Font
- CodeLens

Language-specific settings:
`
    "[al]": {
    "editor.codeLens": false
}
`

## Keyboard Shortcuts
again, a json file.

Make it your familiar editor:
F3 - Insert Line
F4 - Delete Line

## App and launch
app.json
